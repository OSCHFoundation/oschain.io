(static) resolve(domain, optsopt) → {Promise}

Returns a parsed `osch.toml` file for a given domain. Returns a `Promise` that resolves to the parsed osch.toml object. If `osch.toml` file does not exist for a given domain or is invalid Promise will reject. ```js OschSdk.OschTomlResolver.resolve('acme.com') .then(oschToml => { // oschToml in an object representing domain osch.toml file. }) .catch(error => { // osch.toml does not exist or is invalid }); ```
